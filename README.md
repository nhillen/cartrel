# Cartrel

Direct wholesale rails for Shopify stores.

## Core Value Props

**For Brands (Suppliers):**
- Lower fees than marketplaces
- Keep your retailer relationships
- Automated catalog sync across all stockists
- Flexible terms and tier management

**For Retailers (Stockists):**
- Always up-to-date product info and inventory

- Clear view of tier benefits and progress

## How It Works

### 1. Supplier Setup
1. Install Cartrel app in your Shopify store
2. Mark which products are wholesale-eligible
3. Set wholesale pricing
4. Configure default terms and tier rules
5. Invite your retailers

### 2. Retailer Onboarding
1. Receive invite link from supplier
2. Install Cartrel
3. Automatically connect to supplier's catalog
4. See your specific wholesale pricing

### 3. Place Orders
1. Browse supplier catalog in Cartrel app
2. Build purchase order
3. Submit â†’ creates order in supplier's Shopify
4. Pay via supplier's checkout (prepay) or NET terms
5. Track status and fulfillment

### 4. Optional: Import Products
Retailers can import products to their own store:
- Synced inventory from supplier
- Configurable markup rules
- Automatic updates when supplier changes products

## Tech Stack

- **Backend**: Node.js + TypeScript
- **Database**: PostgreSQL + Prisma
- **Queue**: Redis + Bull
- **Frontend**: React + Shopify Polaris
- **Integration**: Shopify Admin API + Webhooks
